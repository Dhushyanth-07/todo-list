# TaskFlow - Professional To-Do List Application

## Overview

TaskFlow is a modern and responsive task management application designed to help users organize, track, and manage daily activities efficiently. The application provides a clean and professional user interface with task tracking capabilities and persistent local storage support.

## Features

* Create and manage tasks easily
* Mark tasks as completed
* Delete tasks when no longer needed
* Real-time task statistics
* Persistent data storage using Local Storage
* Responsive design for desktop and mobile devices
* Professional dashboard-style interface
* Clean and intuitive user experience

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

taskflow/

## Usage

1. Enter a task in the input field.
2. Click the **Add Task** button or press **Enter**.
3. Mark tasks as completed using the checkbox.
4. Remove tasks using the delete button.
5. Task data will be automatically saved in the browser.
